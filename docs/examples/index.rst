Examples
========

The following examples demonstrate the basic usage of the library in all aspects.
Some of the examples are specific to a certain hardware platform or communication protocol, some are generic and can be used with any hardware platform 
and some are specific to use with the `Processing`_ examples.

Ensure that the baud rate in the examples matches the baud rate set in the Arduino IDE's Serial Monitor. Note that some examples require a magnetic field to obtain a valid angle value.
